# Security Policy

## Supported Versions
We release security updates for the following versions of AegisOps AI:

| Version | Supported |
|---------|-----------|
| Latest  | ✅        |
| Older   | ❌        |

## Reporting a Vulnerability
If you discover a security vulnerability in AegisOps AI:

1. **Do not open a public issue.**
2. Email the maintainers at **shreyashtapke@gmail.com** with details.
3. Include:
   - Steps to reproduce
   - Affected modules/files
   - Potential impact
   - Suggested fix (if known)

We will:
- Acknowledge receipt within 48 hours.
- Investigate and confirm the issue.
- Provide updates until resolution.
- Release a patch and update documentation.

## Responsible Disclosure
Please give us reasonable time to fix the issue before public disclosure.  
We follow industry best practices to ensure user safety and privacy.
